MEDIATION data catalog: Methodological developments for a robust, efficient digital twin of the ocean

Welcome to the web data catalog for the MEDIATION project. Here you will find all the information on the data that have been (or are being) used in the project and that are available so far. These data are of various types, such as model namlists, forcings, boundary contitions and outputs, or processing tools.

About the project

The MEDIATION project aims to conduct methodological studies, in order to develop a set of innovative, versatile ocean modelling chains from existing models.
